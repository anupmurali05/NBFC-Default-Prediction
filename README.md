
# NBFC Loan Foreclosure Prediction

This study focuses on understanding and optimizing the factors affecting foreclosure in NBFC loans. There is a need for focus on this issue as NPAs for NBFC loans is increasing at a multi-fold rate. NBFC banks suffers a severe issue of information asymmetry and they are at a disadvantage while lending to borrowers.
